# Data-Analyst-Job-Market-Analytics (SQL)

## Project Overview
This project analyzes data analyst job postings to identify
high-demand and high-paying skills using SQL.

## Tools Used
- MySQL
- MySQL Workbench

## Key SQL Concepts
- Joins
- Aggregations
- CTEs
- Data cleaning

## Key Insights
- SQL is the most in-demand skill for Data Analysts
- Python offers the highest average salary
- Opportunity score helps prioritize skills to learn
